# 学习方法、路径
前半学期电路分析部分，上课听讲以及题库刷题基本可以cover。MOSFET,BJT小信号模型是承上启下的关键部分，首先要理解课内PPT，记住（$g_m=\frac{I_C}{V_T},r_o = \frac{V_A}{I_D}=\frac{1}{\lambda I_D}$）等基本量的公式，把`CS(CE),CG(CB),CD(CC)`等小信号模型自己推导一遍，在理解的基础上记忆。运放、频响、功放部分，课内PPT基本可以cover。反馈部分重在理解，首先要找对反馈组态，再根据组态来得出闭环输入输出电阻与拆环后输入输出电阻的乘除关系（总是想好的方向走），反馈部分推荐B站UP **这个ximo不太冷**。

## 一些物理量的现实意义
理解输入输出电阻的现实意义
- 输入电阻：从信号源索取电流的能力
- 输出电阻：带负载的能力
## 需要记忆的小信号参数
如果$r_o$（一般很大）与$\frac{1}{g_m},\frac{R_B}{\beta+1}$等较小的电阻并联时，直接不考虑（不出现在式子里）

期末考试中会详细说明是否考虑$r_o$

### MOSFET
$g_m = \frac{I_D}{V_{OV}/2}$

$r_o =\frac{1}{\lambda I_D}, 也即 r_o =\frac{V_A}{I_D}，其中 \lambda = \frac{1}{V_A},$ $r_o$为沟道调制效应（厄雷效应）对应的电阻

    MOSFET多级级联的时候，要找到$V_{in}$，一般是G（栅极Gate），才能使用公式和小信号模型

# 频响
- 下限截止频率：低频分析，考虑串联在电路里的电容（看得见的，画出来的），不考虑晶体管的寄生电容
- 上限截止频率：高频分析（交流分析），低频分析时的电容全部短路（电路图里看的见的电容全部短路，即 不考虑串联在电路里的电容（看得见的，画出来的）），考虑晶体管的寄生电容；开路时间常数法只开路三个寄生电容，低频分析时的电容全部短路（电路图里看的见的电容全部短路）始终都是短路

# 波特图
## 相位波特图
画图方法 （第一遍听时较难理解，需刷题库）

从中间往两边画，频率相对大小可以忽略1或者只有1
先确定中频增益，再第一个极点的相位是中频增益对应相位减45度，之后每个极点的相位都为上一个极点减90度（与实际原理相符），（由于每个极点的影响范围是正负一个数量级，因此部分相邻数量级的极点直接相连即可，数量级超过一个的极点用斜率为-1的直线标出该阶段（即正负一个数量级的阶段），然后将各阶段的端点相连即可）
## 幅度波特图
先算出中频增益，每个极点处斜率绝对值增加20dB/decade
