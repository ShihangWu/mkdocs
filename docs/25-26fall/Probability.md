# 学习方法（个人向）
## 课本内容
前三章主要难点在于概率分布函数、概率密度函数的计算，多多练习，提高二重积分的计算能力；大数定律要熟记切比雪夫多项式等，熟悉常见题型即可；统计量、假设检验、参数估计部分，熟记各个量的公式及结论
## 作业与考试
作业要熟练使用计算器，卡西欧999CNCW计算器有统计功能，可以计算平均数、方差、标准差等统计量，分布功能貌似无法计算从负无穷开始积的积分值和积到正无穷的积分制，这一块还是查课本的附表更好。应对考试，首先要把课本例题过一遍，熟悉题型，课本统计量、假设检验、参数估计部分，要牢记各个量的公式，要多练习计算能力
